# 👨‍💻 Portfólio Pessoal | Leandro Rodrigues

Bem-vindo ao repositório do meu portfólio pessoal! Este projeto foi desenvolvido para apresentar minha trajetória profissional, habilidades técnicas e os principais projetos que desenvolvi ao longo da minha carreira e graduação.

[Visualização do portifólio](https://leandroportifolio.vercel.app/)

## 🚀 Tecnologias Utilizadas

Este portfólio foi construído com foco em performance, componentização e design moderno utilizando o ecossistema React:

* **[React.js](https://reactjs.org/)** - Biblioteca JavaScript para construção de interfaces de usuário.
* **[Vite](https://vitejs.dev/)** - Ferramenta de build super rápida para projetos web modernos.
* **[Tailwind CSS](https://tailwindcss.com/)** - Framework CSS Utility-first para estilização ágil e responsiva.
* **[Lucide React](https://lucide.dev/) & [React Icons](https://react-icons.github.io/react-icons/)** - Bibliotecas de ícones SVG.

## ✨ Principais Funcionalidades

* **Design Responsivo:** Interface adaptada perfeitamente para desktops, tablets e smartphones.
* **Dark Mode Nativo:** Esquema de cores focado em conforto visual e estética moderna (Dark UI).
* **Navegação Suave (Smooth Scroll):** Âncoras que deslizam suavemente entre as seções da página.
* **Sistema de Modais:** Detalhamento completo dos projetos em uma janela sobreposta (Modal) sem necessidade de recarregar a página.
* **Galeria Lightbox:** Expansão de imagens e screenshots dos projetos ao clicar, permitindo visualização em tela cheia.
* **Download de CV:** Botão integrado para download direto do currículo em formato PDF.
